# Pre-Class Programming Module

Selamat datang di **Pre-Class Programming Module**! 🚀

Modul ini dirancang khusus untuk peserta yang baru memulai perjalanan di dunia pemrograman (Front-End Web Development). Materi ini disusun secara mendalam namun mudah dipahami untuk membantu kamu membangun fondasi yang kuat.

## 🎯 Tujuan Pembelajaran
- Membangun fondasi kuat dalam pemrograman web.
- Memahami estetika dan tata letak modern (CSS).
- Menguasai logika pemrograman dan manipulasi data (JavaScript).
- Terbiasa menggunakan alat pengembang profesional (CLI, Git, & GitHub).

## 📚 Struktur Kurikulum

### 1. HTML (Bab 1 - 14)
Berisikan dasar-dasar struktur web, penggunaan elemen semantik, pembuatan form, integrasi media, hingga optimasi SEO dasar.

### 2. CSS (Bab 15 - 35)
Mempelajari cara mempercantik tampilan web mulai dari selektor, Box Model, Flexbox, Grid, hingga teknik Responsive Design agar website tampil maksimal di semua perangkat.

### 3. JavaScript (Bab 36 - 57)
Mempelajari logika inti pemrograman JS, mulai dari variabel dan tipe data, fungsi, manipulasi DOM, hingga materi advanced seperti Asynchronous JS dan penggunaan Web API.

### 4. CLI, Git, & GitHub (Bab 58 - 67)
Membekali peserta dengan kemampuan menggunakan Command Line Interface (CLI) dan Version Control System (Git) untuk kolaborasi tim di GitHub.

## 🛠️ Cara Menggunakan Modul
1. Modul dibagi menjadi folder berdasarkan kategori (Section).
2. Setiap bab berada dalam file Markdown (`.md`) yang terpisah.
3. Baca materi secara berurutan mulai dari Bab 1.
